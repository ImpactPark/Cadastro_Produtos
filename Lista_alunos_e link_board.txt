Descrição do Projeto
Projeto: Cadastro de Produtos para inventário - Inventário+ 

Este projeto tem como objetivo o cadastro e geração de relatório para controle de estoque para empresas, tanto do ramo alimentício como eletrônico.

Neste sistema, o usuário deverá ser capaz de executar as seguintes ações:
- Realizar o login através de uma tela inicial
- Se cadastrar caso ainda não tenha acesso ao sistema
- Cadastrar produtos no sistema
- Realizar alterações no estoque quando necessário
- Gerar um relátorio com o inventário dos produtos

O Sistema deverá:
- Armazenar em um banco de dados, os dados dos usuários cadastrados
- Cadastrar e armazenar no banco de dados, produtos do tipo (alimentício, Eletrônico ou Informática)
- Listar os itens cadastrados no inventário
- Ser capaz de alter os produtos, bem como suas quantidades e preços.
- Gerar um relatório em PDF com o inventário.
